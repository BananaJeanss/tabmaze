# tabmaze

a maze game, in your browser, using tab, and some other keys

<https://tabmaze.vercel.app/>

## Controls

- Tab: Next cell
- Shift + Tab: Previous cell
- CapsLk: Up
- Ctrl: Down
- Enter: Enter Exit

> [!WARNING]
> Avoid pressing CTRL + Tab as it may switch browser tabs.

## Development

1. Clone Repo

```bash
git clone https://github.com/BananaJeanss/tabmaze.git
cd tabmaze
```

2. Install Dependencies

```bash
npm i
```

3. Run Vite Dev Server

```bash
npm run dev
```

4. Open Browser
   Open `https://localhost:5173` in your browser to view the app. This uses mkcert to provide a local HTTPS server for PWA testing.
